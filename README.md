# The Zummoning

This is a  prototype of a game where you summon creatures by combining cards, which then proceed to march toward and attack opposing creatures. Win the game by breaking through the opponents defenses!

All cards can either create a creature, or boost the stats of the creature created with another card.

Play at https://rosskarchner.itch.io/the-zummoning-redux
